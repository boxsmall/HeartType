
# HeartType UI设计规范
## Web端设计文档 v1.0

---

# 一、设计原则

## 1.1 核心理念

- **情感化**：体现恋爱测试的甜蜜感和趣味性
- **简洁**：减少干扰，聚焦核心内容
- **可传播**：设计便于截图分享

## 1.2 设计关键词

```
温暖 · 心动 · 轻松 · 趣味
```

## 5.2 首页（Landing）

### 页面结构

```
┌─────────────────────────────┐
│        心动脉冲背景          │
│         (动态粒子)           │
├─────────────────────────────┤
│                             │
│        🏆 HeartType         │
│      "测测你的恋爱人格"       │
│                             │
│    [开始测试]  [排行榜]       │
│                             │
│    ────  features  ────     │
│    💕 快速准确    🎯 趣味    │
│    📊 维度分析    🔗 匹配    │
│                             │
│    ──── 用户评价 ────        │

│                             │
```

### 设计要点

- **背景**：粉色渐变 + 浮动心形粒子动画
- **主标题**：大字号居中，"心动"感
- **CTA按钮**：主按钮粉色渐变，hover放大效果
- **功能展示**：4个icon+文字，2x2布局
- **用户评价**：横向滚动的用户卡片

### 组件规格

| 组件 | 规格 |
|------|------|
| Logo | 48px，粉色心形+文字 |
| 主标题 | H1，36px，居中 |
| 副标题 | 正文，16px，浅灰色 |

### 设计要点

- **动画**：心形跳动，百分比数字滚动

```

- 百分比：数字滚动动画
- 文字：淡入淡出切换
- 整体：1.5-2秒过渡到结果页
```

# 九、无障碍设计

## 9.1 色彩对比

- 文字与背景对比度 ≥ 4.5:1
- 大字号文字对比度 ≥ 3:1

## 9.2 交互

- 所有可点击元素 ≥ 44px × 44px
- 支持键盘导航
- 焦点状态有明显指示

## 9.3 语义

- 使用正确的HTML语义标签
- 图片包含alt描述
- 按钮有明确label
