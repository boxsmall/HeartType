# 一、产品概述

## 1.1 产品定位

HeartType 是一款基于心理学的恋爱人格测试产品，通过50道题目分析用户在恋爱中的10个核心维度，输出人格类型、匹配分析和建议。产品定位为**轻量级情感测试工具**，适合社交传播和广告变现。

## 1.2 核心价值

- **用户价值**：帮助用户了解自己的恋爱人格类型，识别潜在风险，找到合适的伴侣
- **商业价值**：高传播性带来流量，人格标签便于精准广告投放

## 1.3 排名逻辑说明

恋爱价值的评估基于三个核心维度：

```
恋爱价值 = 情绪价值 + 安全感 + 关系稳定性
```

- **S级**：三项都高（天花板伴侣）
- **A级**：2项突出（恋爱体验极佳）
- **B级**：1项强但有短板（可发展但有门槛）
- **C级**：明显缺陷（容易消耗）
- **D级**：高风险/负收益（避雷）

# 三、题目结构设计

## 3.1 维度体系

测试包含 **10大维度 × 5题 = 50题**，每题三个选项（默认计分）：

- A选项：偏负向（1分）
- B选项：中性（2分）
- C选项：偏正向（3分）
- D维度（Q46-Q50）为反向计分：A=3分，B=2分，C=1分

### 10大维度说明

| 维度 | 缩写 | 含义 |
|------|------|------|
| 自我认知 | S | 对自我的认知和评价 |
| 情感表达 | E | 在恋爱中表达情感的方式 |
| 价值观 | A | 对人生、规则的态度 |
| 关系模式 | R | 对亲密关系的偏好 |
| 控制欲 | C | 在关系中的掌控程度 |
| 情绪稳定 | F | 情绪波动的频率和强度 |
| 成就动机 | M | 对事业和成功的追求 |
| 生活方式 | L | 对生活稳定性的偏好 |
| 社交能力 | X | 社交主动性 |
| 依赖程度 | D | 对亲密关系中依赖程度的倾向（分值越高依赖越高） |

## 5.2 维度匹配算法

### 维度关系类型

#### ① 互补（加分）
- C（控制欲）高 + D（依赖）高 → +分
- S（自我）低 + 对方S高 → +分
- F（情绪）波动 + 对方F稳定 → +分

#### ② 相似（稳定加分）
- E（情感）接近 → 加分
- A（价值观）接近 → 加分
- R（关系模式）接近 → 加分

#### ③ 冲突（扣分）
- 双方C（控制欲）都高 → -分
- 双方D（依赖）都高 → -分
- E（情感）差距过大 → -分

---

### 维度评分公式

```
差值 = |A - B|
相似分 = 10 - 差值（差值越小越好，最高10分）
```

---

### 关键维度权重

| 维度 | 权重 | 说明 |
|------|------|------|
| E（情感） | ×1.5 | 核心维度 |
| R（关系） | ×1.5 | 核心维度 |
| F（情绪） | ×1.3 | 重要维度 |
| C（控制） | ×1.2 | 重要维度 |
| D（依赖） | ×1.2 | 重要维度 |
| 其他 | ×1.0 | 一般维度 |

---

### 匹配度计算

```
匹配度 = Σ(维度相似分 × 权重) / Σ(权重)
```
